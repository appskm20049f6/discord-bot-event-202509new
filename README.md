# DC-longtimeBot

這是一個使用最新 discord.js v14 的 Node.js Discord Bot 範例。

## 快速開始

1. 安裝依賴：
   ```powershell
   npm install
   ```
2. 編輯 `index.js`，將 `YOUR_BOT_TOKEN_HERE` 替換為你的 Discord Bot Token。
3. 啟動 Bot：
   ```powershell
   npm start
   ```

## 功能
- 登入事件
- 監聽訊息並回覆 `!ping` 指令
